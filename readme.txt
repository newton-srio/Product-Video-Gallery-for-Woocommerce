== Description ==
Product Video Gallery for WooCommerce allows you to add multiple video URLs (YouTube, Vimeo, or self-hosted) to your product galleries.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit any WooCommerce product and add video URLs in the 'Product Video URL' section.
